SECTION 6 — TECHNICAL EXECUTION RULES
═══════════════════════════════════════════

- Use PptxGenJS (node.js), layout LAYOUT_16x9 (10" × 5.625")
- Safe fonts only: Calibri, Arial (never Aptos)
- NEVER use "#" prefix on hex color strings
- NEVER reuse option objects (shadow, fill) across multiple addShape calls 
  — use a factory function like makeShadow() and makeFill()
- No decorative color bars or accent stripes on cards
- Every text box must have sufficient h (height) to avoid overflow — 
  add 20% slack
- Run visual QA after generation: convert to PDF, render to JPG images, 
  inspect each slide for overflow, overlap, and misalignment
- Fix all visual defects before declaring complete
